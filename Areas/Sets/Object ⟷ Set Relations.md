
# Object  ⟷  Set Relation

<br>

$Object \in Set$

The  <kbd>  Object  </kbd>  is part of the  <kbd>  Set  </kbd>

<br>
<br>

$Object \notin Set$

The  <kbd>  Object  </kbd>  isn't part of the  <kbd>  Set  </kbd>

<br>
<br>

## Examples

$Set_{ A} = \lbrace 1 , 2 , 3 \rbrace$

$1 \in Set_{ A}$

$4 \notin Set_{ A}$
