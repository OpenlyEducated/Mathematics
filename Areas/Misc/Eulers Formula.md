
# Eulers Formula

<br>

$\huge\exp(x) = \frac{ x^{ 0} }{ 0! } + \frac{ x^{ 1} }{ 1! } + \frac{ x^{ 2} }{ 2! } \cdots$

$\huge             =   \frac{ 1 }{ 1 }   +   \frac{ x }{ 1 }  + \frac{ x^{ 2} }{ 2 } \cdots$

$\huge             =     1    +    x   + \frac{ x^{ 2} }{ 2 } \cdots$

$\huge             =     \displaystyle\sum_{n  =  1}^{\infty} \frac{ x^{ n} }{ n! }$
